INTRODUCTION TO HTML

I'll do a brief introduction to HTML, (Hypertext Markup Language),

HTML stands for Hypertext Markup Language.
It is the standard language used to create and structure content on the web.
Everything you see on the web pages (text, images, links, forms) is written using HTML. HTML document contains contents, web browsers read and display . With HTML, we can add everything displayed on a web site.
HTML is practically the skeleton of a website, we can also call it the building blocks of a web pages,
HTML is the foundation of every webpages. Without HTML, web pages wouldn't exist, even when using modern tools and platforms.
HTML is always working in the background. HTML is not a programming language but a Markup Language .

KEY FEATURES OF HTML

  • Structure: HTML organizes content into headings, paragraphs, lists, and sections.
  • Links: It connects paages together using hyperlinks.
  • Media: It allows you to add image, videos, and audio to web pages.
  • Forms: It helps create input fields, buttons, and interactive features.
  • Simple and Universal: Every browser understands HTML.

UNDERSTANDING THE NAME

  • Hypertext: Refers to text that links to other text or web pages.
  • Markup: Means that HTML uses tags to "mark up" or define the different parts of a document.
  • Language: It's a standardized set of rules that browsers understand.

WHAT HTML IS NOT

  • HTML is not a programming language. It doesn't have logic like "if" or "loop".
  • It doesn't make things pretty on its own, that's what CSS is for.
  • It doesn't make things interactive, that's what javascript is for.

    WHAT CAN YOU DO WITH HTML?

  • Structure content on a web page
  • Organize text into headings, paragraphs, and lists
  • Build simple websites or webpages
  • Create your digital resume or portfolio
  • Write structured online articles or tutorials
  • Lay the foundation before styling or adding interactions

INTRODUCTION TO CSS

CSS stands for Cascading style sheets. It is used to style and layout web pages, for example to change the color of text, align items, create spacing,and make your web responsive on different devices.

WHAT IS CSS?

 

CSS is a language that describes how the HTML elements are to be displayed on screen, paper, or in other media.

  • Why do we need CSS?

Without CSS, every website would look like plain, unstyled text, like a word document. CSS allows you to:

  • Change fonts and colors
  • Add spacing and borders
  • Position elements
  • Make responsive designs for mobile devices
  • Add animations and transitions

Basic CSS structures

  • CSS is written using rules called declarations.
  • Each rule targets an HTML eement (called a selector) and applies styles to it.
  • A property is what is being changed (like color or font size).
  • A value is the setting you apply to that property (like red or 16px).

Three ways to Use CSS

There are three ways to use CSS and they are;

1. Inline CSS

  • Style is written directly inside an HTML tag.
  • It is used for quick, one-time changes.
  • Not recommended for large projects.

2. Internal CSS

  • style rules are written inside a style tag in the HTML head section
  • Useful for small websites or when you don't want to create a seperate file.

3. External CSS

  • Styles are written in seperate css file.
  • This file is linked to the HTML.
  • Best practice for large websites.
  • Keeps content (HTML) and designs (CSS) seperate.

What Does "Cascading" Mean?

CSS follows a hierarchy where some rules take priority over others. The order of importance (from highest to lowest)

  1. Inline styles
  2. Internal styles (in the head section)
  3. External styles (linked CSS file)

More specific rules override general ones. The last rule written will override earlier ones if they conflict.

The ID Attribute In CSS

 

What Is An ID?

 
  • An ID is a unique identifier assigned to an HTML element.
  • It helps to identify one specific element on a webpage.
  • An ID must be unique within a page no two elements should have the same ID.

Purpose of the ID attribute

 
  • Used to apply styles to a specific element.
  • Used to target an element for javascript or link anchors.
  • Helps with navigation and scripting by providing a unique reference.

How CSS uses IDs

 
  • In CSS, you use the ID selector to target an element with a specific ID.
  • The ID selector is marked by a hash symbol (#) followed by th ID name.
  • Because IDs are unique, styles applied to an ID only affect that one element.

Example Uses Of IDs

 
  • Styling a single button differently from others.
  • Highlighting a specific section or heading.
  • Linkink directly to a part of the page via an anchor link.
  • Using javascript to manupulate that element specifically.
 

Important Points About IDs

 
  • Each ID value should be unique on a webpage, use IDs carefully.
  • IDs have higher specificity than class selectors, so they override class styles when both apply.
  • If you want to style multiple elements the same way, use classes instead of IDs

Positioning Elements in CSS: CSS allows you to control how elements are positioned on the page using properties like position, top, left, right, and bottom. The most common position values are:

  • absolute: Elements are positioned relative to the nearest positioned ancestor (not static).
  • fixed: Elements are positioned relative to the browser window and stay in place when scrolling.
  • sticky: Elements toggle between relative and fixed, depending on the scroll position.
